全局鉴权请求头 (Authentication Header)

所有 API 请求必须携带以下 HTTP 校验头。复制下方代码时，系统已自动为您替换为了上方生成的最新 API Key（若无则显示默认占位符）：
Authorization: Bearer sk_mqttapi_your_token

GET /api/devices 获取所有物联网设备数据与传感器状态快照 
curl -X GET "http://localhost:22102/api/devices" \
  -H "Authorization: Bearer sk_mqttapi_your_token"
POST /api/devices/:deviceId/relays/:relayId/control 向指定物联网设备下发继电器开/关控制指令 
curl -X POST "http://localhost:22102/api/devices/esp8266_living/relays/relay1/control" \
  -H "Authorization: Bearer sk_mqttapi_your_token" \
  -H "Content-Type: application/json" \
  -d '{"status": "on"}'
GET /api/devices/:deviceId/history 查询指定设备的传感器采样历史时序记录
可选参数 (Query Parameters):
• limit: 获取条数上限，默认 100，允许范围 1 至 500。
• range: 历史时间区间，可选 1h (1小时)、24h (24小时)、7d (7天)。若传入则优先使用时间段筛选且强制上限限制 500 条。 
# 示例 1: 获取最新 100 条历史数据
curl -X GET "http://localhost:22102/api/devices/esp8266_living/history" \
  -H "Authorization: Bearer sk_mqttapi_your_token"

# 示例 2: 查询最近 24 小时内的采样记录并返回
curl -X GET "http://localhost:22102/api/devices/esp8266_living/history?range=24h" \
  -H "Authorization: Bearer sk_mqttapi_your_token"
GET /api/devices 获取所有物联网设备数据与传感器状态快照 
fetch("http://localhost:22102/api/devices", {
  method: "GET",
  headers: {
    "Authorization": "Bearer sk_mqttapi_your_token"
  }
})
.then(res => res.json())
.then(data => console.log(data));
POST /api/devices/:deviceId/relays/:relayId/control 向指定物联网设备下发继电器开/关控制指令 
fetch("http://localhost:22102/api/devices/esp8266_living/relays/relay1/control", {
  method: "POST",
  headers: {
    "Authorization": "Bearer sk_mqttapi_your_token",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({ status: "on" }) // "on" 闭合 / "off" 断开
})
.then(res => res.json())
.then(data => console.log(data));
GET /api/devices/:deviceId/history 查询指定设备的传感器采样历史时序记录
可选参数 (Query Parameters):
• limit: 获取条数上限，默认 100，允许范围 1 至 500。
• range: 历史时间区间，可选 1h (1小时)、24h (24小时)、7d (7天)。若传入则优先使用时间段筛选且强制上限限制 500 条。

  method: "GET",
  headers: {
    "Authorization": "Bearer sk_mqttapi_your_token"
  }
})
.then(res => res.json())
.then(data => {

  console.log(data);
});
